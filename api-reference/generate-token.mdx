---
title: "Token"
description: "Generate a token to authenticate requests"
api: "GET https://api.zingfi.co/token"
authMethod: "none"
---

## Parameters
<ParamField query="api_key" type="string" required>
  Copy your API Key from your dashboard
</ParamField>

## Response Fields

<ResponseField name="token" type="string">
The bearer token you need to run authenticate other requests
</ResponseField>

<ResponseField name="expires_in" type="integer">
The expiration time for the token
</ResponseField>

<ResponseField name="token_type" type="string">
The type of token
</ResponseField>



<ResponseExample>

```json Status 200
{
  "token": "eyJhbGciOiJBMjU2S1ciLCJlbmMiOiJBMjU2Q0JDLUhTNTEyI.....",
  "expires_in": 3600,
  "token_type": "Bearer"
}
```

```json Status 403
{
  "code": "ERROR_CODE_ACCESS_DENIED",
  "message": "Invalid API key",
  "payload": ""
}
```

</ResponseExample>
