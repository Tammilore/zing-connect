---
title: "Get Provider Codes"
description: "Fetch all provider codes"
api: "GET https://api.zingfi.co/extract/providers"
---

## Response Type

<ResponseField name="name" type="string">
      The name of the provider
    </ResponseField>

    <ResponseField name="code" type="string">
      The provider's code
    </ResponseField>

<ResponseExample>

```json 200
[
  {
    "name":"Access Bank",
    "code":"044"
  },
  {
    "name":"GTBank",
    "code":"50211"
  },
  {
    "name":"Kuda Bank",
    "code":"50211"
  }
  ...
  ...
    ]
```

</ResponseExample>
