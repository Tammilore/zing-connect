---
title: "Create Job"
description: "Start an extraction job"
api: "POST https://api.zingfi.co/extract/start"
---

## Parameters

<ParamField body="base64_string" type="string" required>
  The base 64 string of the document
</ParamField>

<ParamField body="code" type="string" required>
  The code of the provider the document is from
</ParamField>

## Response Type

<ResponseField name="message" type="string">
      A generic message about the extraction
    </ResponseField>

    <ResponseField name="status" type="string">
      The status of the job
    </ResponseField>
    
    <ResponseField name="job_id" type="string">
      The job_id for the document
    </ResponseField>

<ResponseExample>

```json Status 200
{
  "message": "Document processing started",
  "status": "STARTED",
  "job_id": "2f546..."
}
```

</ResponseExample>
